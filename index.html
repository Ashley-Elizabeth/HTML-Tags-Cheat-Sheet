HTML Tags Cheatsheet for Beginners

META

Tag Name Description
<!doctype html> doctype declaration* Tells the computer that it's looking at an HTML page. Put this at the top of your HTML document.
<meta charset="utf-8"> meta charset* Tells the computer that you are using the UTF-8 character set (which is the character set for most of the world wide web). Guards against your site turning into gibberish.
<title>..</title> title* Gives your site a title in the top of the browser bar.
<script src="http://www.example.com/scripts/myjavascript.js">..</script> src attribute Links to external JavaScript files via the src attribute
<link ="stylesheet" href="http://www.example.com/css/style.css"> href & rel attributes Links to external stylesheets via the href attribute. Use the rel attribute to tell the computer that this is a stylesheet.

STRUCTURAL

Tag Name Description
<html> </html> html* Denotes the HTML portion of the document.
<head> </head> head* Denotes the head area of your site where the meta tags go. None of this conent is visable to the user.
<body></body> body* Denotes the body area of your document. Everything inside these tags is visible to the user.

CONTENT

Tag Name Description
<div> </div> division The workhorse of the HTML tags. Use divs for your different site sections, sidebars, footers, etc.
<p>...</p> paragraph Denotes the head area of your site where the meta tags go. None of this conent is visable to the user.
<img src="http://www.example.com/image/image.jpg" alt="My Image" /> image** Adds images to your site. Put the image link in the src attribute and the name in the alt attribute.
<ul>...</ul> unordered list Unordered list, perfect for bulleted lists and such.
<ol>...</ol> ordered list Ordered lists, good for numbered or alphabetical lists.
<li> </li> list item Each list item goes inside of an li tag.
<a href="http://www.example.com">...</a> href Adds links to external sites via the href attribute.
<span>...</span> span Span tags allow you to ground inline elements.
<strong>...</strong> strong Use to make sections of your text bold.
<em>...</em> emphasis Use these to make portions of your text italic.

HEADLINES

Tag Name Description
<h1> </h1> heading 1 Most important headline. Great for logos, page titles, etc.
<h2> </h2> heading 2 Second most important headline. Use for article headlines or headlines for major sections of the page.
<h3>...</h3> heading 3 Third most important headline. Use for less important articles or other secondary content.
<h4>...</h4> heading 4 Fourth most important headline. Good for sidebars and footers.
<h5>...</h5> heading 5 Fifth most important heading.
<h6>...</h6> heading 6 Sixth most important heading.

TABLES

Tag Name Description
<table> </table> table Use this tag to add tables to your site.
<thead> </thead> table head Denotes the table head, where you put your columns and names.
<th> </th> th Use these tags inside your table head to give each column a name.
<tbody> </tbody> table body Denotes the table body, where all the tabular data goes.
<tr> </tr> table row Use this tag to denote a table row.
<td> </td> table data Put each piece of data inside a data cell, inside of your table row, insude of your table body.
